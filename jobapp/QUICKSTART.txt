STEP 1: PRE-REQUISITES (1 minute)
========================================

Ensure you have:
✅ XAMPP installed (Apache + MySQL)
✅ Project files in: C:\xampp\htdocs\jobapp\
✅ Web browser (Chrome, Firefox, Edge)
✅ phpMyAdmin access

========================================
STEP 2: START SERVICES (1 minute)
========================================

1. Open XAMPP Control Panel
2. Click START for:
   - Apache   (port 80)
   - MySQL    (port 3306)
3. Wait for both to show green status

========================================
STEP 3: CREATE DATABASE (1 minute)
========================================

Method A: Using phpMyAdmin (Easiest)
1. Open: http://localhost/phpmyadmin/
2. Click "New" button
3. Enter database name: jobapp
4. Click "Create"
5. Select "jobapp" database
6. Click "Import" tab
7. Choose file: project.sql (from jobapp folder)
8. Click "Go"
→ Database created with all tables, views, procedures, triggers

Method B: Using MySQL Command Line (Alternative)
```
mysql -u root
CREATE DATABASE jobapp;
USE jobapp;
SOURCE C:\xampp\htdocs\jobapp\project.sql;
```

STEP 8: QUERY TESTING IN phpMyAdmin (3 minutes)
========================================

Test Complex Queries:

1. STORED PROCEDURES
   SQL Tab → Execute these:
   
   CALL total_applicants();
   → Shows: total applicants, unique emails, birth months

   CALL get_applicants_by_position('Software Engineer');
   → Shows: all applicants for Software Engineer position

   CALL get_recruitment_statistics();
   → Shows: overall statistics and metrics

2. VIEWS
   
   SELECT * FROM applicant_details_view;
   → Full applicant information with all relationships
   
   SELECT * FROM submission_summary_view;
   → Aggregated statistics per position

3. COMPLEX JOINS
   
   SELECT A.name, S.position, L.city, V.vision_name, B.experience
   FROM Applicant A
   JOIN Submissions S ON A.applicant_id = S.applicant_id
   JOIN Locate L ON A.applicant_id = L.applicant_id
   JOIN Background B ON A.applicant_id = B.applicant_id
   LEFT JOIN Visions V ON S.submission_id = V.submission_id;
   → Full data from all 5 tables with JOINs

========================================
TROUBLESHOOTING
========================================

Issue: Database not found
→ Solution: Ensure project.sql was imported

Issue: Connection failed
→ Solution: Check MySQL is running in XAMPP

Issue: Form not submitting
→ Solution: Clear browser cache, check console for errors

Issue: Procedures not showing
→ Solution: Re-import project.sql, refresh phpMyAdmin

Issue: Triggers not working
→ Solution: Check XAMPP MySQL version supports triggers

Issue: Pages not loading
→ Solution: Check Apache is running, URL is correct
